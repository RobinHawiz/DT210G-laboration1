# DT210G-Laboration1

Målet med denna labb är att skapa en enkel webbplats med React och TypeScript uppdelade i lämpliga komponenter. Webbplatsens syfte är att representera något fritidsintresse jag har, tex böcker.

Utseende på webbplatsen bestämmer man själv men ska minst innehålla:

- Ett sidhuvud innehållande titel på sidan samt en banner innehållande en bild. Titel ska skickas som en prop till komponenten.
- Sidfot med valfritt innehåll, tex vilken som är utvecklare av webbplatsen.
- Huvudinnehåll bestående av en lista med dynamiskt genererade komponenter baserat på data. Använd map-funktion för att iterera över en array med objekt. Minst tre objekt ska finnas i listan.
